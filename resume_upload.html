Instructions for Uploading Your Resume

  • Accepted File Formats: PDF (.pdf), Microsoft Word (.doc, .docx), Text File (.txt)
  • File Size Limit: Maximum size of 10 MB
  • File Naming: Use your full name and position (e.g., JohnDoe_SoftwareEngineer.pdf)
  • File Quality: Ensure the document is clear and readable. PDFs should not be password protected or scanned images.
